# Todo-manager
Personal Todo manager

## Functional Requirements
- Add new tasks with: 
  - Title 
  - Description 
  - due date
  - End date
  - Status  
  - start date 
- Edit existing tasks
- Mark status as completed, inprogress, on-hold and pending.


#User Management
- create user with
    - UserName
    - email
    - password
    - Last name

#validation
task title should be minimum 5 word
Due date should be greater than current date
