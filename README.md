# Todo-manager
Personal Todo manager

## Functional Requirements
- Add new tasks with: 
  - Title 
  - Description 
  - due date
  - Start date
  - Status
  -  UserName
- Edit existing tasks
- Mark status as completed, inprogress and pending.


## User Management
- create user with
    - UserName
    - email


## Task create validation
task title should be minimum 5 word
Due date should be greater than current date
